📌 Project Title

Exploratory Data Analysis (EDA) on Books Dataset

🎯 Objective

The goal of this project is to analyze a books dataset and find useful insights using Python.

📂 Dataset Information

The dataset contains:

Book Title
Price
Availability
Rating
🛠 Tools Used
Python
Pandas
Matplotlib
Seaborn
📊 Steps Performed
Created dataset using Python
Explored data structure
Cleaned and transformed data
Converted ratings into numbers
Performed statistical analysis
Created visualizations
📈 Visualizations

The following graphs were created:

Price Distribution
Rating Distribution
Boxplot (Outliers)
Scatter Plot (Price vs Rating)
Correlation Heatmap
📌 Key Insights
Most books are moderately priced
Ratings are mostly between 3 and 5
No strong relationship between price and rating
Data is clean and simple
🚀 How to Run This Project
1. Install libraries
pip install pandas matplotlib seaborn
2. Run Python file
python eda_project.py
